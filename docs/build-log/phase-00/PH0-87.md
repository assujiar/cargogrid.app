## 5. Real bugs found and fixed during authoring (not left open)

1. **Stale local `main` ref caused a false collision positive.** First run of `check-worktree-collision.ts` flagged `claude/sleepy-ride-8pg1em` as a genuine fork of `agent/cargogrid-autonomous-build` — 49 "commits ahead of main." Root cause: local `main` (`53e3d4a`) was 6 commits behind `origin/main` (`715cc08`) in this sandbox. Fixed by comparing against `origin/main`, not local `main`; verified with `git merge-base origin/main claude/sleepy-ride-8pg1em` = that branch's own HEAD (i.e. fully contained, zero real divergence). Also refined the collision predicate itself: "more than one branch ahead of a ref" is not the same as "a genuine fork" — a branch that is a plain ancestor of another (unmoved pointer) is not a collision. Added pairwise `git merge-base` fork detection (`isGenuineFork`) and a regression test pinning this exact fix.
2. **`.env` protected-path regex also matched `.env.example`.** `checkProtectedPaths` flagged the repository's own safe, tracked `.env.example` file as `FORBIDDEN`. Fixed with a negative lookahead excluding `.example`/`.sample`/`.template` suffixes; verified against 9 real path variants (`.env`, `.env.local`, `.env.production`, `.env.development.local` → forbidden; `.env.example`/`.sample`/`.template`, including nested paths → allowed).
